# 软工导论知识图谱的应用
软工导论课程项目，前端使用vue2 + elemnetui进行开发，后端采用Java(Maven进行项目管理), 数据库使用neo4j

## 前端
前端对应的文档在front/application目录下的README中

## 后端

### Neo4jTest

首先用maven进行构建，安装依赖包

需要在本地运行以下命令

~~~
neo4j windows-service install
neo4j start
~~~

**Requires:**

~~~
jdk = 17.0.8
Neo4j = 5.24.2 (社区版)
~~~
